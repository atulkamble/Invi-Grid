# 🚀 AWS + Azure + DevOps – 200 Interview Q&A

---

# 🔹 SECTION 1: CLOUD FUNDAMENTALS (1–25)

### 1. What is Cloud Computing?

Delivering computing services (servers, storage, DB, networking) over the internet.

### 2. Types of Cloud?

Public, Private, Hybrid, Multi-cloud

### 3. IaaS vs PaaS vs SaaS?

* IaaS → Infra (EC2, VM)
* PaaS → Platform (App Service)
* SaaS → Software (Office 365)

### 4. What is High Availability?

System remains operational even if one component fails.

### 5. What is Fault Tolerance?

System continues without downtime even after failure.

### 6. What is Scalability?

Ability to increase/decrease resources.

### 7. Vertical vs Horizontal Scaling?

* Vertical → Increase CPU/RAM
* Horizontal → Add more servers

### 8. What is Load Balancer?

Distributes traffic across multiple servers.

### 9. What is CDN?

Content Delivery Network (low latency delivery).

### 10. What is Region & AZ?

* Region = geographical location
* AZ = isolated datacenter

### 36. What is AMI?

### 37. What is Auto Scaling?

### 38. What is ELB?

### 39. What is Amazon RDS?

Managed relational database service.

### 40. What is DynamoDB?

NoSQL DB service.
